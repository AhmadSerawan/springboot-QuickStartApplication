## Maven Lifecycle Practice

This project demonstrates core Maven build phases using :contentReference[oaicite:0]{index=0}:

### Commands executed:

- `mvn clean` → removes previous build files
- `mvn compile` → compiles source code
- `mvn package` → creates JAR/WAR file
- `mvn verify` → runs tests and validates build

### Purpose:
To understand how Maven builds a project step-by-step from source code to packaged artifact.
